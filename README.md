## 概要
2進、10進、16進数を相互に変換します。

```
変換対象の数
基数
変換先の基数
```
の標準入力で動作します。
```
11111111
2
16
```
と標準入力で渡すと、
```
11111111(2)を16進数に変換
FF(16)
```
 と標準出力で返します。
16進数でアルファベットを含む場合は大文字のみ対応です。
## 工夫した点と所感
対応しない数や変換対象を入れた際の例外処理や、複数の基数に変換するための例外処理を追加して穴がないように出来たと思います。<br>
初めに設計しないで行き当たりばったりで書き始めてしまったのでおそらく冗長な部分があります。<br>
当初の予定より規模が大きめのコードになってしまいました。最初に丁寧に処理を考えたり規模を想定してから製作すべきと学びました。<br>
また、使用するメソッドなどを調べていた際、Integerクラスの
```
Integer.parseInt(str, base)
Integer.toString(n, base)
```
メソッドを知りました。<br>
実務ではそちらを使用すると思うので有益性はあまりありませんが、コーディングの学習目的なので良しとします。<br>
